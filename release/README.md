# Moon Outside Map Tablet

Adds a tablet-style schematic map that shows the **ship position** and **outside entrances**
(Main Entrance and Fire Exit) on the current moon.

Designed to work with **vanilla and most modded moons**.

---

## Features

- Tablet-style map opened with a hotkey
- Ship-centered schematic view
- Main entrance (green) and Fire Exit (red)
- Works on modded moons (Wesley Moons, single-moon mods, etc.)
- Configurable rotation and scale
- Does NOT open while typing in Terminal or UI fields

## Configuration

Config file is generated at:

BepInEx/config/


Options include:
- Toggle key
- Map scale (meters per pixel)
- Map rotation offset
- Dot size

---

## Notes

- The map is **schematic**, not a real terrain map
- Rotation offset is adjustable per player preference
- Some heavily custom moons may not expose entrance data correctly
